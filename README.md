# check-late-app

안드로이드 지각 벌칙 어플

## 내용

친구들과의 모임이나 동아리 등에서 늦는 친구에게 벌칙을 주는 어플이다. 벌칙은 직접 정할 수가 있고 작성한 벌칙에 따라 룰렛을 돌리는 게임을 진행할 수도 있다.
